Introdução

Bem-vindo à documentação da API! Aqui você encontrará informações sobre os endpoints disponíveis e como utilizá-los para interagir com o sistema.

Listar Todas as Tabelas

O endpoint /api/list-all-tables permite obter uma lista com todos os nomes de tabela presentes no banco de dados. Para utilizar este endpoint, faça uma requisição POST com os seguintes parâmetros:

  • host: Host do banco de dados.
  • port: Porta do banco de dados.
  • username: Nome de usuário para acessar o banco de dados.
  • password: Senha para acessar o banco de dados.
  • database: Nome do banco de dados.
  • dialect: Dialeto do banco de dados (por exemplo, "postgres" ou "mysql").

Exemplo de Requisição


POST /api/list-all-tables HTTP/1.1
Content-Type: application/json

{
  "host": "localhost",
  "port": 5432,
  "username": "admin",
  "password": "secretpassword",
  "database": "mydatabase",
  "dialect": "postgres"
}

Parâmetros de Configuração

  • host: Host do banco de dados.
  • port: Porta do banco de dados.
  • username: Nome de usuário para acessar o banco de dados.
  • password: Senha para acessar o banco de dados.
  • database: Nome do banco de dados.
  • dialect: Dialeto do banco de dados (por exemplo, "postgres" ou "mysql").

Resposta

  • success: Indica se a solicitação foi bem-sucedida.
  • tables: Array contendo os nomes das tabelas presentes no banco de dados.

Outros Endpoints

Esta API possui outros endpoints para realizar operações como criação, atualização, exclusão e consulta de dados no banco de dados. Consulte a documentação completa para obter mais detalhes.
