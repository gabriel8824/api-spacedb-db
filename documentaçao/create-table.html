Documentação da API

Introdução

Bem-vindo à documentação da API! Aqui você encontrará informações sobre os endpoints disponíveis e como utilizá-los para interagir com o sistema.

Criar Tabela

O endpoint /api/create-table permite criar uma nova tabela no banco de dados. Para utilizar este endpoint, faça uma requisição POST com os seguintes parâmetros:

  • host: Host do banco de dados.
  • port: Porta do banco de dados.
  • username: Nome de usuário para acessar o banco de dados.
  • password: Senha para acessar o banco de dados.
  • database: Nome do banco de dados.
  • dialect: Dialeto do banco de dados (por exemplo, "postgres" ou "mysql").
  • tableName: Nome da tabela a ser criada.
  • tableComment: Comentário da tabela.

Exemplo de Requisição


POST /api/create-table HTTP/1.1
Content-Type: application/json

{
  "host": "localhost",
  "port": 5432,
  "username": "admin",
  "password": "secretpassword",
  "database": "mydatabase",
  "dialect": "postgres",
  "tableName": "users",
  "tableComment": "Tabela de usuários"
}

Parâmetros de Configuração

  • host: Host do banco de dados.
  • port: Porta do banco de dados.
  • username: Nome de usuário para acessar o banco de dados.
  • password: Senha para acessar o banco de dados.
  • database: Nome do banco de dados.
  • dialect: Dialeto do banco de dados (por exemplo, "postgres" ou "mysql").

Parâmetros da Tabela

  • tableName: Nome da tabela.
  • tableComment: Comentário da tabela.

Resposta

  • message: Mensagem de sucesso indicando que a tabela foi criada com sucesso.
